# Titanic Dataset - Exploratory Data Analysis (EDA)

## 📘 Overview
This project performs in-depth Exploratory Data Analysis (EDA) on a sample Titanic dataset to uncover insights and patterns in the data.

## 📊 Dataset
The Titanic dataset includes details such as passenger age, class, fare, sex, and survival status.

## 🧪 EDA Steps
- Data inspection and cleaning
- Missing value analysis
- Statistical summary
- Distribution visualization (histograms, KDE)
- Outlier detection (boxplots)
- Correlation analysis (heatmaps)

## 📂 Structure

## ▶️ How to Use
1. Open and explore visuals in the `visuals/` folder.
2. Run your own analysis using the dataset in `data/`.
